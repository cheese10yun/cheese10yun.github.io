Spring JPA Best Practices step-16 - QuerydslлҘј мқҙмҡ©н•ҙм„ң Repository нҷ•мһҘн•ҳкё° (2)

2019/06/13

н•ҙлӢ№ мҪ”л“ңлҠ” GithubлҘј нҷ•мқён•ҙмЈјм„ёмҡ”.

JpaRepositoryмқҳ мҝјлҰ¬ л©”м„ңл“ңлҘј нҶөн•ҙм„ң к°„лӢЁн•ң мҝјлҰ¬л“Өмқ„ м•„лһҳ мҳҲм ң мІҳлҹј мүҪкІҢ л§Ңл“ӨмҲҳ мһҲмҠөлӢҲлӢӨ.

мң мӮ¬н•ң мҝјлҰ¬к°Җ н•„мҡ”н•ҙм§Җл©ҙ мҝјлҰ¬ л©”м„ңл“ңлҘј м§ҖмҶҚм ҒмңјлЎң м¶”к°Җн•ҙм•ј н•ҳлҠ” лӢЁм җмқҙ мһҲмҠөлӢҲлӢӨ. мқҙлҹ° кІҪмҡ°м—җ QuerydslPredicateExecutorлҘј мӮ¬мҡ©н•ҳл©ҙ л§Өмҡ° нҡЁкіјм Ғмһ…лӢҲлӢӨ.

QuerydslPredicateExecutor#

QuerydslPredicateExecutor мҪ”л“ңмқҳ мқјл¶Җмһ…лӢҲлӢӨ. PredicateлҘј л§Өк°ңліҖмҲҳлЎң л°ӣкі  мһҲкё° л•Ңл¬ём—җ PredicateлҘј нҶөн•ҙм„ң мғҲлЎңмҡҙ мҝјлҰ¬лҘј л§Ңл“ӨмҲҳ мһҲмҠөлӢҲлӢӨ.

AccountRepository м Ғмҡ©н•ҳкё°#

AccountSupportRepositoryлҠ” step-15: QuerydslлҘј мқҙмҡ©н•ҙм„ң Repository нҷ•мһҘн•ҳкё°(1) м—җм„ң м¶”к°Җн•ң мҪ”л“ңмқҙкі  QuerydslPredicateExecutor<Account> мҪ”л“ңл§Ң м¶”к°Җн•ҳл©ҙ мҷ„лЈҢ лҗ©лӢҲлӢӨ.

QuerydslPredicateExecutor мӮ¬мҡ©н•ҳкё°#

AccountRepositoryлҠ” QuerydslPredicateExecutorлҘј кө¬нҳ„н•ҳкі  мһҲмқҢмңјлЎң лі„лӢӨлҘё мҪ”л“ң м—Ҷмқҙ мҡ°лҰ¬лҠ” AccountRepositoryлҘј нҶөн•ҙм„ң QuerydslPredicateExecutorмқҳ л©”м„ңл“ңлҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙкІғмқҖ мҡ°лҰ¬к°Җ AccountRepositoryк°Җ м•„л¬ҙ л©”м„ңл“ңк°Җ м—Ҷм–ҙлҸ„ JpaRepositoryм—җ мһҲлҠ” findById, findAll к°ҷмқҖ л©”м„ңл“ңлҘј мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мқҙмң мҷҖ лҸҷмқјн•©лӢҲлӢӨ.

Test Code#

Predicate к°қмІҙ мғқм„ұмқ„ нҶөн•ҙм„ң мҝјлҰ¬ л©”м„ңл“ң мҪ”л“ң м¶”к°Җ м—Ҷмқҙ лӢӨм–‘н•ң мҝјлҰ¬л“Өмқ„ мүҪкІҢ л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.

predicate_test_001 н…ҢмҠӨнҠё мҪ”л“ңм—җ лҢҖн•ң мӢӨм ң мҝјлҰ¬ лӮҙмҡ©мһ…лӢҲлӢӨ. н•ҙлӢ№ мҝјлҰ¬к°Җ м–ҙл–»кІҢ м¶ңл ҘлҗҳлҠ”м§Җ м§Ғм ‘ лҲҲмңјлЎң нҷ•мқён•ҳлҠ” мҠөкҙҖмқ„ к°–лҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

кІ°лЎ #

мЎ°нҡҢмҡ© мҝјлҰ¬лҘј л§Ңл“ңлҠ” кІғмқҖ мӢӨм ң нҳ„м—…м—җм„ң л§ҺмқҖ м—…л¬ҙ л№„мӨ‘мқ„ м°Ём§Җн•ҳлӢӨ ліҙлӢҲ мһ‘м—…мһҗл“ӨмқҖ лӢӨм–‘н•ң мЎ°нҡҢ м „мҡ© кө¬нҳ„мІҙл“Өмқ„ мҡ°нӣ„мЈҪмҲңмІҳлҹј л§Ңл“ӨкІҢ лҗ©лӢҲлӢӨ. мқҙл ҮкІҢ лҗҳл©ҙ мӨ‘ліө мҪ”л“ңк°Җ л§Һмқҙ л°ңмғқн•ҳкІҢ лҗҳкі , кө¬нҳ„мІҙк°Җ лҠҳм–ҙлӮҳлҠ” л§ҢнҒј мқҳмЎҙ кҙҖкі„лҸ„ мһҗм—°мҠӨл Ҳ лҶ’м•„м§ҖкІҢ лҗ©лӢҲлӢӨ.

мқҙлҹ¬н•ң л¬ём ңлҘј мң„мҷҖ к°ҷмқҙ к°қмІҙм§Җн–Ҙ кҙҖм җмңјлЎң н’Җм–ҙлӮҙм–ҙ AccountRepository мқён„°нҺҳмқҙмҠӨлҘј нҶөн•ҙм„ң DAOк°Җ м ңкіөлҗҳкі  м„ёл¶Җ кө¬нҳ„л“Өмқ„ мҲЁкёё мҲҳ мһҲкІҢ м„Өкі„н•ҳлҠ” кІғмқҙ л°”лһҢм§Ғн•ҳлӢӨкі  мғқк°Ғн•©лӢҲлӢӨ.

лӢЁмҲңн•ҳкІҢ JPAлҘј мһҳ нҷңмҡ©н•ҳлҠ” кІғліҙлӢӨ мң„м—җм„ң м–ёкёүн•ң лӮҙмҡ©мқ„ мЎ°кёҲ лҚ” к№ҠкІҢ мғқк°Ғн•ҙліҙлҠ” кІғмқҙ мқҙ нҸ¬мҠӨнҢ…м—җ м·Ём§Җмһ…лӢҲлӢӨ.
